- Micro-interactions for icons (wishlist, cart, compare)

### Pages Included

#### Primary Pages
1. **Home Page** - Hero banner, featured collections, new arrivals, best sellers, testimonials
2. **Shop Page** - Category-based product grid with filters and sorting
3. **Product Detail Page** - Image gallery with zoom, size selector, add to cart/wishlist/compare
4. **Login/Sign Up Page** - Glassmorphism UI with animated inputs
5. **Cart Page** - Product list with quantity controls and price summary
6. **Compare Page** - Side-by-side product comparison table
7. **Wishlist Page** - Saved products grid with quick add-to-cart
8. **Checkout Page** - Address form, order summary, payment options (UPI, Card, COD)
9. **Order Confirmation Page** - Success animation and order summary
10. **User Account/Dashboard** - Profile info, order history, edit profile
